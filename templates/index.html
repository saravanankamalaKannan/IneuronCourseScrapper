API User Guide:

"/" - lauch the home page(current page)

"/courses" - returns the json object with list of available Courses

"/course/valid_course_name" - returns the json object with list of available Sub-Courses ex: "/courses/Cloud"

"/courses/detail" - returns the json object with status as success or failure and stores the scrapped details in mongo DB
